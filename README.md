## Part B: Concepts
- **IP Address:** Unique number identifying my device on the network
- **MAC Address:** Hardware ID of my network card — 00-50-56-C0-00-08
- **Default Gateway:** My router's IP — 192.168.0.1 — routes traffic to internet
- **DNS:** Translates domain names to IPs — handled by my router 192.168.0.1
- **Public vs Private IP:** 192.168.174.1 is private (local only);
  Public IP is assigned by ISP and visible on internet
